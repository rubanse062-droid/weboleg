# Litewin

Static landing page (Vancouver, WA appliance repair).

## Deploy on Vercel
1. Put ALL files at the ROOT of the repo/upload (index.html must be in the root, not in a subfolder).
2. Framework Preset: **Other**. Build Command: empty. Output Directory: empty (or `.`).
3. Deploy.

Works the same on Netlify / GitHub Pages / any static host.
